  <!-- GETTING STARTED -->
  ## Getting Started
    
  If you would like to clone the repo hopefully this will help.
    
  ### Prerequisites
    
  rust, npm
    
  ### Installation
    
  1. Clone the repo
        ```sh
        git clone https://github.com/imjord/rust_react_simple_full_stack.git
        ```
  2. Install NPM packages in client folder
        ```sh
        npm install
        ```
  3. Cargo run/ Cargo build inside server folder
        ```sh
        cargo run
        ```
  4. Run the start command in client
        ```sh
        npm start 
        ```
  5. Enjoy! made with <3 by imjord
  <p align="right">(<a href="#readme-top">back to top</a>)</p>
